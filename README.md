# Electricity-Energy-usage-tracker
⚡ Smart Electricity Usage Monitor  The Smart Electricity Usage Monitor is a responsive web application designed to help users track, manage, and optimize their electricity consumption. Built using HTML, CSS, and JavaScript, this project promotes energy awareness and cost savings through interactive monitoring and real-time insights.
The application allows users to set a monthly electricity limit, update current usage in kWh, and calculate their estimated bill based on the electricity rate per unit. It provides instant feedback on consumption status and alerts users when they approach or exceed their predefined usage threshold. The built-in notification system enhances awareness and encourages responsible energy usage.

To improve data visualization, the project integrates Chart.js for displaying dynamic line charts that represent usage trends over time. Users can also view a detailed usage history table with timestamps, export data in CSV format, remove the last entry, or reset all stored data.

Additional features include billing period tracking, daily budget calculations, projected monthly cost estimation, dark mode toggle, and practical electricity-saving tips. The app uses browser localStorage to preserve data even after page refresh.

This project demonstrates strong front-end development skills, including DOM manipulation, data visualization, and client-side storage implementation.
